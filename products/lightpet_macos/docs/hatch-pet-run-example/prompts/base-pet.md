# Base Pet Prompt Example

Create the canonical base reference for a LightPet-compatible desktop pet.

Package metadata:

- id: `lulu`
- displayName: `Lulu`
- rendering: `smooth`

Visual identity:

- Small rounded capybara desktop pet.
- Compact readable silhouette.
- Soft toy-like 3D form with simple warm colors.
- Transparent or clean chroma-key-ready background.

Output purpose:

This generated base becomes `decoded/base.png` and
`references/canonical-base.png`. Every row-strip job must use it as the visual
source of truth.
