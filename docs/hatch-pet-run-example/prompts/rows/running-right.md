# Running-Right Row Prompt Example

Generate the `running-right` animation row for the same pet.

Requirements:

- 8 frames in one horizontal strip.
- Preserve the canonical base identity, silhouette, palette, face, proportions,
  material feel, and rendering style.
- Show rightward drag or locomotion through body and limb movement only.
- Do not draw speed lines, dust clouds, shadows, motion trails, text, UI,
  frame numbers, guide marks, or detached effects.

Inputs for the real job:

- `references/canonical-base.png`
- `references/layout-guides/running-right.png`
