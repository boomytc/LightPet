# Idle Row Prompt Example

Generate the `idle` animation row for the same pet.

Requirements:

- 6 frames in one horizontal strip.
- Preserve the canonical base identity, silhouette, palette, face, proportions,
  material feel, and rendering style.
- Show a calm resting loop with subtle breathing or blinking.
- Keep the pet centered inside each 192x208 frame slot.
- Do not include text, UI, frame numbers, guide marks, detached effects,
  shadows, floor patches, or scenery.

Inputs for the real job:

- `references/canonical-base.png`
- `references/layout-guides/idle.png`
